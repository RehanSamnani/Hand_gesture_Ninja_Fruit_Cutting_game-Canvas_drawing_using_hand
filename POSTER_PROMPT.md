# POSTER DESIGN PROMPT - Hand Gesture Interactive Application

## PROJECT OVERVIEW

**Project Name:** Hand Gesture Interactive Application (Ninja Fruit Game + Drawing Mode)

**Description:** A Python-based computer vision application using real-time hand gesture recognition via MediaPipe and OpenCV to control two interactive game modes without any physical controller.

**Project Type:** Computer Vision + Game Development + Human-Computer Interaction

**Key Innovation:** Natural hand gesture-based control with 21-point hand landmark detection using cutting-edge AI technology

## DESIGN PRINCIPLES

### Visual Hierarchy
1. **Level 1 (Dominant):** Project name + main visual (50-60% focus)
2. **Level 2 (Strong):** Tagline + main features (25-30% focus)
3. **Level 3 (Supporting):** Technical specs + CTA (15-20% focus)

### Emphasis Techniques
- Use bold colors (cyan, neon green) for key elements
- Glowing effects around hand and gesture indicators
- Subtle shadows and depth for 3D appearance
- White space to prevent cluttering
- Color contrast for readability (dark text on light, or vice versa)

### Audience Appeal
- **For Gamers:** Show exciting game screenshots with vibrant colors
- **For Developers:** Highlight technical achievements (MediaPipe, 30-60 FPS, real-time)
- **For Students:** Emphasize educational value (AI, CV, hand detection)

## TIPS FOR MAXIMUM IMPACT

1. **Keep it Simple:** Maximum 3 colors + black/white
2. **Make it Bold:** Use large, bold text for headlines
3. **Show Action:** Use dynamic images of the game running
4. **Create Contrast:** Ensure text stands out from background
5. **Include QR Code:** Link directly to GitHub or setup instructions
6. **Use Whitespace:** Don't overcrowd - breathing room is important
7. **Add Movement:** Consider animated elements for digital displays
8. **Test Readability:** Readable from 10+ feet away for physical posters
9. **Brand Consistency:** Use consistent design language throughout
10. **Call-to-Action:** Make it clear what the viewer should do next
