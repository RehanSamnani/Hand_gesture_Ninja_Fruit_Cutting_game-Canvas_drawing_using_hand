# 🚀 START HERE - Ninja Fruit Game with Hand Gestures

## 👋 Welcome!

You've found the **Hand Gesture Interactive Application** - an amazing game where you control the action with your hand! No controller needed, just your webcam and hand gestures.

---

## ⚡ Get Started in 3 Steps

### Step 1️⃣: Clone the Project
```bash
git clone https://github.com/RehanSamnani/Hand_gesture_Ninja_Fruit_Cutting_game-Canvas_drawing_using_hand.git
cd Ninja-Fruit-Like-Game-with-hand-gesture-and-opencv
```

### Step 2️⃣: Run Setup
**Windows:**
```bash
setup.bat
```

**macOS/Linux:**
```bash
python setup.py
```

### Step 3️⃣: Play!
```bash
python "Ninja Fruit with hand gesture.py"
```

**That's it! The game should now be running! 🎮**

---

## 🎮 What Can You Do?

### Option 1: Play Ninja Fruit Game 🥝🍎
- **Objective**: Cut falling fruits before they hit the ground
- **Control**: Use your hand - move it to move the cursor, touch fruits to cut them
- **Score**: 100 points per fruit
- **Difficulty**: Increases as you play
- **Lives**: Start with 15 lives
- **Animation**: See beautiful cutting effects when you score!

**Press 1** from the main menu to play.

### Option 2: Draw with Your Hand ✏️🎨
- **Drawing Modes**: 
  - Lines (Press L)
  - Circles (Press O)
  - Free drawing (Press F)
- **Colors**: Red, Green, Blue, Yellow, Cyan, Magenta, White, Black
- **Instructions**: Press H to see all shortcuts
- **Clear**: Press C to clear canvas

**Press 2** from the main menu to draw.

| **O** | Circle mode |
| **F** | Free drawing |

## 💡 Tips for Best Experience

### Hand Detection
✅ **Good lighting** - Bright room helps detection  
✅ **Show your full hand** - Keep hand visible and unobstructed  
✅ **Distance** - Position hand 1-2 feet from camera  
✅ **Single hand** - App works best with one hand at a time

### Fruit Game
✅ **Quick movements** - Fast swipes catch more fruits  
✅ **Stay centered** - Remain in middle of screen  
✅ **Smooth control** - Steady, controlled motions work best

### Drawing Mode
✅ **Clear pinch** - Make sure thumb and index touch  
✅ **Slow drawing** - Draw slowly for cleaner lines  
✅ **Bright background** - Helps hand detection

### Quick Issues?
1. **Game won't run**: Make sure setup completed successfully
2. **Camera not working**: Grant camera permission to Python
3. **Hand not detected**: Increase lighting, show hand clearly
4. **Drawing not working**: Pinch more clearly (thumb touches index)
