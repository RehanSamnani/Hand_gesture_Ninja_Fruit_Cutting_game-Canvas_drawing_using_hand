## 🎮 Game Features Quick Reference

### Game Mode (Press 1)
```
Controls:  Hand = Cursor | Touch = Cut
Scoring:   100 points per fruit
Lives:     Start with 15
Difficulty: Increases every 1000 points
Features:  ✨ Cutting animations, 📊 Live stats
```

### Drawing Mode (Press 2)
```
Gestures:  Open = Move | Pinch = Draw
Shapes:    L=Line, O=Circle, F=Free
Colors:    R=Red, G=Green, B=Blue, Y=Yellow
Tools:     H=Help, C=Clear, M=Menu
```
