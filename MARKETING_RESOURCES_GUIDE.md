## 🎯 HOW TO USE THESE PROMPTS

### For Creating the Poster:

**Step 1: Choose Your Tool**
- Canva (beginner-friendly, many templates)
- Adobe Illustrator/InDesign (professional)
- GIMP (free, open-source)
- Microsoft Publisher
- Online tools (Piktochart, Venngage)

**Step 2: Select Design Style**
- Review POSTER_PROMPT.md layouts
- Choose between vertical/horizontal
- Pick a color scheme
- Select poster version (Event/Educational/Social)

**Step 3: Gather Visual Assets**
- Take screenshots of your game
- Capture hand gesture images
- Collect technology logos
- Create or download icons
- Prepare any artwork

**Step 4: Build the Poster**
- Start with background and color scheme
- Add main visual/image (50-60% of space)
- Add headline and tagline
- Include key features with icons
- Add call-to-action and links
- Include system requirements

**Step 5: Finalize and Export**
- Check readability from distance
- Verify all text is visible
- Export as PDF (print) and PNG (digital)
- Test on different screens
- Get feedback from others

---

### For Creating the PowerPoint:

**Step 1: Set Up Your Presentation**
- Create new PowerPoint/Google Slides
- Set to 16:9 aspect ratio
- Apply consistent color scheme
- Add slide numbers and date
- Choose professional fonts

**Step 2: Build Slide by Slide**
- Start with Title Slide
- Copy content from POWERPOINT_PROMPT.md
- Add visuals/screenshots to each slide
- Keep to 1 main idea per slide
- Maximum 5 bullet points per slide

**Step 3: Add Visual Elements**
- Include project screenshots
- Add technology logos
- Insert architecture diagrams
- Include performance charts
- Add hand detection visualizations

**Step 4: Prepare for Presentation**
- Practice delivery 2-3 times
- Time your presentation
- Test demo section thoroughly
- Prepare speaker notes
- Create backup copies
- Test on projector/display

**Step 5: Deliver Effectively**
- Make eye contact with audience
- Speak confidently and clearly
- Let visuals do the talking
- Show live demo effectively
- Encourage questions
- Follow up on interested contacts
